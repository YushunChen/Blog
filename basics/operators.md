# Operators

## Types of Operators

* Arithmetic Operators
* Assignment Operators
* Comparison \(Relational\) Operators
* Logical Operators
* Bitwise Operators

### Arithmetic Operators

```go
package main
import "fmt"
func main() {
    var a int=20
    var b int=11
    fmt.Println(a+b)
    fmt.Println(a-b)
    fmt.Println(a*b)
    fmt.Println(a/b)
    fmt.Println(a%b)
}
```
